# CoreInventory - Smart Inventory Management System

## Architecture
- **Backend**: Express.js + MongoDB + JWT + Socket.io
- **Frontend**: Next.js 14 (App Router) + Tailwind + ShadCN + Framer Motion + Three.js

## Implementation Plan
1. Backend setup & all APIs
2. Frontend landing page with 3D animations
3. Dashboard with full CRUD operations
4. Real-time features & polish
